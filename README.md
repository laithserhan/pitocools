# pitocools

_Pico tools_

Template engine for pico source code.

## Usage

```shell
pitocools build src.p8

```

Use any of the following command to build your games:

- __lua(filename)
- __gfx(filename)
- __gff(filename)
- __map(filename)
- __sfx(filename)
- __music(filename)

To include the corresponding section of the corresponding file in the generated p8 file.
